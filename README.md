# 🚀 AI Image to Notes, Audio & Quiz

An AI-powered Streamlit app that converts images into **structured notes, audio explanations, and quizzes** based on difficulty.

---

## 🌐 Live Demo
👉 https://imgto-notes-and-quiz.streamlit.app/

---

## ✨ Features

- 🖼️ Upload **images**
- 🧠 Generate **AI-based notes**
- 🎧 Convert notes into **audio (Text-to-Speech)**
- ❓ Generate **quizzes with selectable difficulty**
- ⚡ Simple UI built with Streamlit

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- gTTS (Text-to-Speech)  
- Gemini Api

---

## 🚀 How It Works

1. Upload images  
2. AI generates notes from images  
3. Notes are converted into audio  
4. Quiz is generated based on difficulty
